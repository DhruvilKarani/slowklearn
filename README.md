# slowklearn
An sklearn library clone library made for fun.

## Structure

```
    .
    |-- loss    #various loss functions
    |-- metrics
    |-- supervised
    │   |-- linear
    |       |-- LinearRegression
    |       |-- LogisticRegression
    │   |-- tree
    │   |-- test    # Unit tests
    |                 
    |-- unsupervised
        |-- kmeans
        |-- word2vec
```
